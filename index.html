¿Qué es la búsqueda informada?

La búsqueda informada es un tipo de algoritmo que utiliza conocimiento específico del dominio (heurísticas) para guiar el proceso de búsqueda hacia soluciones óptimas de manera más eficiente.

En este simulador, implementamos el algoritmo A* donde:

  • f(n) = g(n) + h(n) - La función de evaluación
  • g(n) - El costo acumulado hasta el nodo actual
  • h(n) - La heurística que estima el costo al objetivo

Estructuras de Datos en Búsquedas

Las estructuras de datos utilizadas varían según el tipo de búsqueda:

Tipo de Búsqueda Estructura de Datos Característica
Amplitud (BFS) Cola (FIFO) Explora niveles completos antes de avanzar
Profundidad (DFS) Pila (LIFO) Explora un camino hasta el final antes de retroceder
Costo Uniforme Cola de Prioridad Ordena por costo acumulado g(n)
Greedy Cola de Prioridad Ordena por valor heurístico h(n)
A* Cola de Prioridad Ordena por f(n) = g(n) + h(n)

En nuestro caso: El algoritmo A* utiliza una cola de prioridad que organiza los estados según su valor f(n), seleccionando siempre el de menor valor para expandirlo.
